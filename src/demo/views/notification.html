基本使用
基本使用，从右上角弹出，默认显示5s。 
//成功通知
notification({
    type:'success',
    title:'成功提示',
    msg:'成功内容'
})

//信息通知
notification({
    type:'info',
    title:'信息提示',
    msg:'信息内容'
})

//警告通知
notification({
    type:'warning',
    title:'警告提示',
    msg:'警告内容'
})

//错误通知
notification({
    type:'error',
    title:'错误提示',
    msg:'错误内容'
}))
不自动消失的通知
通过设置time=0，使通知不会自动消失，用户可点击X进行关闭。 
notification({
    type:'success',
    title:'成功提示',
    msg:'我不会自动消失',
    time:0
})
自定义按钮通知
当需要在通知框内有额外操作时，可使用confirmBtn，可自定义按钮上的事件，此回调函数有一个参数，该参数是一个方法，用来关闭通知，如果需要点击后关闭通知，在onClick回调中执行该方案；closeBtn不允许自定义事件，该按钮作用等同于X。 
notification({
    title:'成功提示',
    msg:'请点击下方按钮',
    time:0,
    confirmBtn:{
        show:true,
        text:'点我确认',
        onClick:( close )=>{
            message({
                type:'success',
                msg:'可以在这里执行自定义事件'
            });
            //关闭通知
            close();
        }
    },
    closeBtn:{
        show:true,
        text:'点我关闭'
    }
})
