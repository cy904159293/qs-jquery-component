基本使用
基本使用，初始化modal时不设置title和确认回调，每次调用modal时再根据需要设置，做到modal的复用。 
this.confirmModal = new ConfirmModal({});

const that = this;
that.confirmModal.setData({
    title:'确认删除',
    msg:'是否确认删除本条信息？',
    confirmFn:()=>{
        that.confirmModal.modalHide();
        message({
            type:'success',
            msg:'删除成功！'
        })
    }
}).modalShow();
