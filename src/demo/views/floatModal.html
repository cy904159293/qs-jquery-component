基本使用
基本使用，4中方向。 
this.floatModal2 = new FloatModal({ noInput:true });
const that = this;
that.floatModal2.show({
    title:'向上的弹框',
    obj:$(this),
    side:1,
    fn:()=>{
        that.floatModal2.hide();
        message({
            type:'success',
            msg:'确认向上的弹框'
        })
    }
})

带输入框的弹框
用来要求用户输入一个信息来验证。 
this.floatModal2 = new FloatModal({});
const k = 4234;
that.floatModal.show({
    title:'请输入'+k+'进行确认',
    obj:$(this),
    side:1,
    placeholder:'请输入验证数字',
    fn:(value)=>{
        if(value-0 === k){
            message({
                type:'success',
                msg:'验证正确'
            });
            that.floatModal.hide();
        }else {
            that.floatModal.showMsg('输入不正确')
        }
    }
})
