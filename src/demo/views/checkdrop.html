基本使用
基本使用。 
this.checkDrop = new CheckDrop({
    obj:$('#test1'),
    renderLi:row=>row.name,
    list:new Array(20).fill(1).map((v,i)=>{
        return {
            name:'选项'+i,
            id:i
        }
    }),
    onSelectChange:ids=>{
        message({
            type:'info',
            msg:'当前选中下标：'+ids.join(',')
        })
    }
});
动态设置选项
异步设置选项，通过renderList方法进行设置。该方法传入[]时，进行清空选项。 
this.checkDrop2 = new CheckDrop({
    obj:$('#test2'),
    renderLi:row=>row.name,
    onSelectChange:ids=>{
        message({
            type:'info',
            msg:'当前选中下标：'+ids.join(',')
        })
    }
});
const that = this;
//设置选项
that.checkDrop2.renderList(new Array(20).fill(1).map((v,i)=>{
    return {
        name:'选项'+i,
        id:i
    }
}));
message({
    type:'success',
    msg:'设置选项成功'
});
//清空选项
that.checkDrop2.renderList([]);
message({
    type:'success',
    msg:'清空选项成功'
})
