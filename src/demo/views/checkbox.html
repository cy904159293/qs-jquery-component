基本使用
基本使用。 
new CheckBox({
    obj:$('#test1'),
    onChange:(checked)=>{
        message({
            type:'info',
            msg:checked?'选中':'未选中'
        })
    }
});
不同颜色
通过改变color参数，调整checkbox颜色；
通过设置checked参数，设置checkbox初始选中状态 
const colors = ['#ff8b00','#00b793','#d52740'];
$('#colorTest').find('.test-check').each((i,target)=> {
    new CheckBox({
        obj:$(target),
        color:colors[i],
        checked:i%2
    })
});
设置/获取选中状态
通过调用setChecked方法，设置组件的选中状态
通过调用getChecked方法，获取组件的选中状态 
this.check = new CheckBox({
    obj:$('#test2'),
    onChange:(checked)=>{

    }
});

const that = this;
//选中
that.check.setChecked(true);

//不选中
that.check.setChecked(false);

//获取选中状态
const checked = that.check.getChecked();
message({
    type:'info',
    msg:'当前选中状态：'+(checked?'选中':'未选中')
})
